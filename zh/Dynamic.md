# 动态

{: .fs-9 }

动态 NFTs 要稍微更复杂些，采用压缩于 html 有效载荷的一个 iframe 来为最终载荷。

动态元素来自于 NFT 的内部程序，该程序能从 Arweave 网关中检索信息，能根据最新合约状态来修改它们的样态。

这仅仅是一个应用示例，但能够提供与现有 Atomic NFT 标准交互所需要的默认功能。
